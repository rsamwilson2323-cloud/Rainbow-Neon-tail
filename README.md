# 🌈 Rainbow Neon Tail

An interactive HTML5 Canvas animation that creates vibrant rainbow-colored neon trails following your touch or mouse movement. The project combines glowing particle effects, smooth fading animations, and dynamic color cycling to create a mesmerizing visual experience.

## ✨ Features

* 🌈 Dynamic rainbow color transitions
* ✨ Glowing neon trail effects
* 🖱️ Mouse interaction support
* 📱 Mobile touch support
* 🎨 Smooth particle fading animations
* ⚡ Real-time Canvas rendering
* 🖥️ Responsive fullscreen display
* 🚀 Lightweight and dependency-free

## 🎮 How to Use

### Desktop

* Click anywhere to create a colorful neon splash.
* Hold and drag the mouse to generate continuous rainbow trails.

### Mobile

* Tap the screen to create glowing color bursts.
* Touch and drag to paint vibrant rainbow trails.
